# Indian Investment Instruments — Founder Reference Guide

Use this when building Section 3 (Retirement & Wealth Preservation) and Section 2 (Tax Planning).

## Tier 1: Tax-Advantaged Instruments (Always Start Here)

### PPF — Public Provident Fund
| Feature | Detail |
|---------|--------|
| Annual Limit | ₹500 (min) to ₹1,50,000 (max) |
| Lock-in | 15 years (partial withdrawal from 7th year; loan from 3rd–6th year) |
| Returns | ~7.1% p.a. (government-set quarterly; currently FY24-25) |
| Tax Status | **EEE** — Exempt at invest (80C), Exempt on growth, Exempt at withdrawal |
| Best For | Long-term wealth foundation; guaranteed, risk-free, fully tax-free |
| Platform | Post office, SBI, HDFC, ICICI, online via net banking |
| Founder Note | Ideal even if income is low — ₹500/month keeps the account active and compounds beautifully |

### NPS — National Pension System (Tier 1)
| Feature | Detail |
|---------|--------|
| Contribution Limit | No upper limit; ₹50,000 gets extra 80CCD(1B) deduction (over 80C) |
| Lock-in | Until age 60 (40% must be annuitised at maturity; 60% lump sum tax-free) |
| Returns | Market-linked; equity NPS historically ~10–12% p.a. |
| Tax Status | **EET** — Exempt at invest (80C + 80CCD), Exempt on growth, Taxable at 40% annuity |
| Best For | Long-horizon retirement savings with higher equity allocation |
| Platform | eNPS (enps.nsdl.com), Zerodha, HDFC, Axis |
| Founder Note | Choose Tier 1 Active Choice with maximum 75% equity allocation for early-stage founders with long horizons |

### NPS Tier 2
| Feature | Detail |
|---------|--------|
| Lock-in | **None** — fully liquid |
| Returns | Market-linked (same funds as Tier 1) |
| Tax Status | No deduction; gains taxed at slab rate (unlike Tier 1) |
| Best For | Liquid parking of surplus with equity upside; not for long-term tax planning |
| Founder Note | Good for accumulating short-term surplus while earning better than FD returns |

### ELSS — Equity Linked Saving Scheme
| Feature | Detail |
|---------|--------|
| Annual Limit | No upper limit; 80C deduction on up to ₹1.5L invested |
| Lock-in | 3 years (shortest among 80C instruments) |
| Returns | ~12–14% CAGR historically (equity mutual fund, not guaranteed) |
| Tax Status | **ETE** — Exempt at invest (80C), Taxable gains (LTCG > ₹1.25L @ 12.5%), Exempt partial |
| Best For | Founders who want 80C benefit with market-linked returns and shortest lock-in |
| Platform | Zerodha Coin, Groww, MFCentral (always buy DIRECT plan) |
| Founder Note | SIP into ELSS makes the 3-year lock-in rolling and manageable. Prefer Flexi-cap ELSS. |

## Tier 2: Liquid & Emergency Fund Instruments

### Liquid Mutual Funds
| Feature | Detail |
|---------|--------|
| Returns | ~6.5–7.5% p.a. (better than savings account ~3–4%) |
| Liquidity | T+1 withdrawal (instant redemption up to ₹50,000 via some platforms) |
| Risk | Very low (invests in overnight/short-term debt instruments) |
| Tax | STCG taxed at slab rate if held < 3 years; LTCG 20% with indexation if > 3 years |
| Best For | Emergency fund parking; better than letting money sit in savings |
| Recommended Funds | Parag Parikh Liquid Fund, HDFC Liquid Fund, Axis Liquid Fund (all direct plan) |
| Platform | Zerodha Coin, Groww, AMC website |

### Flexi Fixed Deposits (Bank FDs)
| Feature | Detail |
|---------|--------|
| Returns | ~6.5–7.5% p.a. (varies by bank and tenure; small finance banks offer ~8–9%) |
| Liquidity | Premature withdrawal possible (0.5–1% penalty on interest) |
| Tax | Interest fully taxable at slab rate; TDS at 10% if interest > ₹40,000/yr |
| Best For | Capital preservation, short-term parking (6–24 months) |
| Founder Note | Use for the "non-negotiable" portion of emergency fund. Sweep-in FDs in salary account are convenient. |

### Savings Account (High-Yield)
| Returns | ~3–4% (regular banks) | ~6–7% (small finance banks like AU, Jana, ESAF) |
| Best For | Daily operational float only — not for emergency fund beyond 1 month's burn |

## Tier 3: Long-Term Wealth Building (Post Layers 1 & 2)

### Index Funds (Passive Equity)
| Feature | Detail |
|---------|--------|
| Recommended Indices | Nifty 50, Nifty Next 50, Nifty 500 |
| Expense Ratio | 0.1–0.2% p.a. (direct plan — critical to avoid regular plan which eats 0.5–1% more) |
| Returns | ~12% CAGR historical (10-year rolling average, Nifty 50) |
| Tax | LTCG > ₹1.25L @ 12.5% (held > 12 months); STCG @ 20% (held ≤ 12 months) |
| Best For | Core long-term portfolio — simple, low-cost, market-returns |
| Top Funds | UTI Nifty 50 Index, Nippon Nifty 50, Motilal Oswal Nifty Next 50 (all direct plan) |
| Platform | Zerodha Coin, Groww, AMC direct website |
| Founder Note | A founder's first ₹5,000 SIP should be in a Nifty 50 index fund. Zero research overhead. |

### Flexicap / Multicap Mutual Funds (Active Equity)
| Feature | Detail |
|---------|--------|
| Returns | ~13–16% CAGR historically (active funds with good track record) |
| Expense Ratio | 0.5–0.8% p.a. (direct plan) |
| Tax | Same as index funds — LTCG/STCG |
| Best For | Supplement to index funds for alpha-seeking, long horizon (7+ years) |
| Top Funds (Direct) | Parag Parikh Flexi Cap, HDFC Flexi Cap, Kotak Flexicap |
| Founder Note | Keep simple — 1 index fund + 1 flexicap fund is more than enough for most founders |

### Sovereign Gold Bonds (SGB)
| Feature | Detail |
|---------|--------|
| Returns | Gold price appreciation + 2.5% p.a. fixed interest |
| Lock-in | 8 years (exit allowed after 5th year on interest payment dates) |
| Tax | **Capital gains fully tax-exempt** if held to maturity |
| Best For | Gold exposure without physical gold; tax-efficient |
| Founder Note | Better than physical gold or Gold ETF. Buy during RBI issuance windows (check RBI website). |

## Tier 4: Insurance (Non-Negotiable Foundation)

### Term Life Insurance
| Feature | Detail |
|---------|--------|
| Recommended Cover | 10–15× annual income or ₹1Cr minimum (whichever is higher) |
| Premium (Indicative) | ₹8,000–15,000/yr for ₹1Cr cover at age 28–35 (non-smoker) |
| Tax | Premium deductible u/s 80C (within ₹1.5L limit) |
| Best For | Founders with dependents — spouse, children, parents reliant on income |
| Top Insurers | HDFC Life Click 2 Protect, Tata AIA Sampoorna Raksha, LIC Tech Term |
| Founder Note | Buy ONLINE direct (not through agent) — 30–40% cheaper. Pure term, no ULIP or endowment. |

### Health Insurance
| Feature | Detail |
|---------|--------|
| Recommended Cover | ₹10L minimum; ₹15–25L in metros; Top-up plan for ₹50L+ at low cost |
| Premium (Indicative) | ₹12,000–25,000/yr for ₹10L family floater at age 30–35 |
| Tax | Premium deductible u/s 80D (₹25,000 self + ₹25,000 or ₹50,000 for parents) |
| Founder Note | Startup group health cover is NOT portable. Buy individual/family plan on day one of founding. |
| Top Insurers | Niva Bupa, Care Health, HDFC ERGO Optima Restore |

### Personal Accident & Critical Illness (Optional but Recommended)
| Best For | Founders with dependents and high-risk travel; disability risk |
| Premium | ₹3,000–8,000/yr for ₹25L–50L cover |

## What to Avoid (Common Indian Founder Traps)

| Instrument | Problem | Better Alternative |
|-----------|---------|-------------------|
| LIC Endowment / Money Back | Low returns (4–5%), opaque, expensive | Term insurance (pure) + ELSS SIP |
| ULIPs | High charges (8–12% in early years), lock-in, poor returns | Direct mutual funds |
| Physical Gold / Jewellery | Making charges, storage risk, not liquid | SGBs or Gold ETF |
| Real estate (early stage) | Illiquid, high EMI stress, kills founder runway | Index fund SIPs for 5+ years first |
| Chit Funds | Unregulated risk, returns not guaranteed | Liquid MF or RD |
| Regular plan mutual funds | Distributor commission eats 0.5–1% extra every year | Direct plan on Zerodha Coin / Groww / AMC |
